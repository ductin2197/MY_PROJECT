## Project Overview
Dự án phân tích hơn 500,000 giao dịch bán lẻ để tìm ra xu hướng doanh thu.
- **SQL:** Làm sạch dữ liệu và xử lý các giao dịch lỗi.
- **Tableau:** Trực quan hóa dữ liệu với các chỉ số quan trọng (85% doanh thu đến từ UK).
